How to upload on GitHub

  • Create a GitHub repository, for example my-blog.
  • Upload this file and rename it to index.html if you want it to open as the homepage.
  • Go to Settings → Pages → choose the branch, then save to publish with GitHub Pages.
  • Later, edit the file directly on GitHub with the pencil icon, or edit it locally in VS Code and upload again.
